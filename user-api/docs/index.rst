Bem vindo à documentação do User Api!
=====================================
User Api é um projeto para manter usuários.


Introdução
===========
Aqui é possível cadastrar, listar, deletar e alterar usuários. O microsserviço também pode ser consumido por outros microsserviços que
tenham os mesmos usuários cadastrados.
