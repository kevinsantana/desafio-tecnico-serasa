Bem vindo à documentação do Order Api!
=======================================
Order Api é um projeto para manter pedidos.


Introdução
===========
Aqui é possível cadastrar, listar, deletar e alterar pedidos. O microsserviço User Api é consultado para relacionar um pedido a um usuário.
Desta forma é possível consultar os pedidos feitos por um usuário específico.
